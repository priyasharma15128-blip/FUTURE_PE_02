# FUTURE_PE_02
# AI-Powered UGC Ad Content Pack for GlowRx Vitamin C Serum

## Project Overview

This project was developed as part of the Future Interns Prompt Engineering Internship – Task 2: AI Content Marketing Using UGC Ads.

The objective was to create a complete AI-powered UGC (User Generated Content) Ad Content Pack for a real product, GlowRx Vitamin C Serum. The project demonstrates how prompt engineering can be used to generate authentic, platform-specific marketing content that helps brands improve engagement and conversions.

## Features

* Multiple scroll-stopping UGC hooks for different marketing angles
* Platform-specific ad scripts for Instagram Reels, YouTube Shorts, and Meta Ads
* High-converting call-to-action (CTA) examples
* Ready-to-use Instagram marketing caption
* Reusable prompt templates for content generation
* Interactive and responsive web interface
* Copy-to-clipboard functionality for easy content reuse

## Technologies Used

* HTML5
* chatgpt
* Prompt Engineering
* AI-Assisted Content Generation

## Project Outcome

The project showcases how structured prompts can generate professional marketing content suitable for brands, agencies, and content creators. It provides a reusable framework for creating UGC advertisements, marketing scripts, hooks, captions, and CTAs across multiple platforms.
